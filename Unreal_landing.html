General description

These projects have all been done with blueprints, except for those that state otherwise.

Uniat: Veracruz Edition

This game is a third person shooter focused on map exploration. The player character fights with a water gun that uses niagara particle system with collisions.

    Content I worked on:
  • Automatic water gun
  • Water particles
  • Implementing animations (aim offset and blend spaces)
  • Inventory system
  • Collectable items
  • Progression keys
  • Gated progression
  • Reloading individual tanks for the weapon
  • Feature Highlight

Astro Bites

This is a restaurant manager game with spaceship ambiance. The gameplay is divided into 3 different stages, where you can collect ingredients and machine upgrades, then cook using machines that transform given ingredients into full dishes.

    Content I worked on:
  • Machines in kitchen using class inheritance
  • Complex data structures for recipes
  • Interaction between data structures and in game objects
  • Simple inventory for machines
  • In world UI elements
  • Feature Highlight

Side Scroller

This game is a procedurally generated 2.5D side scroller where movement is the most important mechanic, with both saving momentum and using a grappling hook.

    Content I worked on:
  • Procedural generation of levels
  • Grappling hook mechanic
  • Smoothing player movement
  • Extra lives
  • Simple enemy
  • Saving level information
  • Feature Highlight

Pablito crafter

This is a 2D sandbox game with survival elements like temperature, hunger, health, a day-night cycle and enemy waves at night time. The enemies are varied and challenging for the player, making him have to use his full toolset of guns and building to eliminate or evade them.

    Content I worked on:
  • Wave system
  • Progressive difficulty
  • Enemy spawners that change spawn probability
  • Waves and day - night cycle integration
  • Player movement
  • Grappling hook
  • Feature Highlight
