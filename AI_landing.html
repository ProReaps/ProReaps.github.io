Genetic algorithm

This is a simulation made in unreal engine of a population of randomly generated individuals trying to achieve a specific body part combination, the more fit individuals are used as bases for the next generation. There's a way to control the mutation value in the simulation.

Fuzzy Logic

This is a demo that uses fuzzy logic to grade a user's aim and comment on their performance. It's set up as an aim trainer with a simple task, to complete the grading process the user must 'shoot' 20 target spheres and after running some functions in the background, there is a simple score screen that shows up.

Expert systems

This is a recreation of the game Wordle with a hint of mastermind's answer grading interface, as in there are no hints on where the correct letters or incorrectly placed letters are. An expert system designed to play the game attempts to guess the correct word and has 10 tries to do so. The system has a list of possible answers and there is also a handy counter to aid in visualizing how it works.

  • Video demo

Guess who binary tree

This is a demo with an AI player that guesses the character you get assigned at the start of the simulation. It uses a binary tree that has each possible guess and a path to all of them.
