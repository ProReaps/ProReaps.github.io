Beach shaders

This is a shader demo for many effects done in unity's shadergraph.

    Content:
  • Player trail buffer
  • Vertex displacement for most objects in the scene
  • Shaders with global and local variables
  • Partially transparent elements
  • Stylized water
  • Vertex displacement with trail buffer for the sand
  • Simulated wind displacement
  • Demo Effect

Earth Shader

This is a shader demo for an animated world effect.

    Content:
  • Animated rotation with desync for clouds
  • Specular lighting mapped to the oceans and water
  • Night mode with different lighting
  • Normal map that influences lighting
  • Scene shadows
  • Demo Effect

Hiding UFO shader (Approved by David Ville)

This is a shader demo for an animated dynamically hiding UFO ship, it uses noise to make the dividing line between the hidden and visible part of the ship.

    Content:
  • Simulated transparent shader
  • Refraction in invisible segment
  • Normal map affects the refraction
  • Invisible segment blurs what is behind it
  • Emissive transition between invisible and visible segments of the ship
  • Noise transforms transition line
  • Transition line is animated
  • Rimlight
  • Emissive ship lights
  • Scene skybox
  • Cubemap is reflected by surface
  • Illumination modified by cubemap
  • Normal map affects reflections and cubemap illumination
  • Demo Effect
